# Installation

Clone this repository into your project:

```bash
git clone https://github.com/arthanson/django-cities-xtd.git
```

Download the zip file and unpack it:

```bash
wget https://github.com/arthanson/django-cities-xtd/archive/master.zip
unzip master.zip
```

Install with pip:

```bash
pip install django-cities
```
